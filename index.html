To get started with Sass, install the gem from the command line.

gem install sass

Sass is going to preprocess your sass code and magically turn it into CSS. But first you need to tell it which file to take in and where to put the result.

sass input.sass output.css

1

Variables

You can assign variables for things you are going to use over and over by starting them with a '$'. This works really well for colors, font stacks.

$teal: #43cea2

$main-font: 'Poppins', sans-serif

h1
font-family: $main-font

This way, changing a color or font would involve changing just one line of the CSS.

Fun fact: underscores and hyphens can be used interchangeably. For example $main-width can access $main_width and vice versa.

2

Nesting

Sometimes you might want to use the nested rule's parent selector using the & character. For example: hovering on a link

a
 color: white
 &:hover
  color: $pink

Example Link

Navigation is another great place for nesting styles.

nav
  ul
    list-style: none

  li
    display: inline-block

  a
   text-decoration: none

3

Partials

Some features lend themselves naturally to being pulled out into another file, as I've done here with the button.

You will need to do two things:

  1. 1. make a new file that begins with an underscore

    _button.sass

  2. 2. import this file into your main SASS stylesheet

    @import button.sass

Order is important here, so make sure you are loading things like colors first.

4

Import

As you've probably noticed, CSS gets pretty unruly pretty fast. The @import feature allows the file to be broken up. And actually, SASS will figure out the file extensions for you so you don't need to add '.sass'.

@import reset
@import typography
@import button

5

Mixins

Mixins are particularly handy for vendor prefixes. Unfortunately all browsers prefer slightly different syntax, but mixins make it easy to DRY this up.

=border-radius($radius)
 -webkit-border-radius: $radius
 -moz-border-radius: $radius
 -ms-border-radius: $radius
 border-radius: $radius

and specify this where you need it, passing in a radius

.code
 +border-radius(5px)

6

Extend/Inheritance

You can also extend the properties of one class to another class. In this case we have a class="button" and a class="button-pink" but we can DRY up our code and just use one class.

.button-pink
 @extend .button

.button-pink:hover
 color: $pink
 border: 3px solid $pink
